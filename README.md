# Yet another project boilerplate

Contents:

- TypeScript
- Fastify
- Pino
- PostgreSQL
- node-pg-migrate
- pgtyped
- React
- Tailwind
- Vite
- Docker
- Testcontainers
- Jest
- Swagger
- Lefthook
- Turborepo

To start:

- Node 16.13.2+
- Yarn 1.22.5+
- `yarn install` in repo root
- `docker compose up --build` to start the dev db
- `yarn dev` to start frontend and backend with Turborepo
- Open [localhost:3000](http://localhost:3000)
